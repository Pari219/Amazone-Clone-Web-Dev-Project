# 🛒 Amazon Clone

A responsive frontend clone of the Amazon homepage built using **HTML5** and **CSS3**. This project is created to practice modern web development concepts, improve CSS layout skills, and replicate the UI of one of the world's largest e-commerce platforms.

---

## 📌 Features

- 🏠 Amazon-inspired homepage
- 📱 Responsive design
- 🧭 Navigation bar with search box
- 📦 Product category sections
- 🖼️ Hero banner
- 🛍️ Product cards with images
- 📄 Footer similar to Amazon
- 🎨 Clean and organized UI

---

## 🛠️ Built With

- HTML5
- CSS3
